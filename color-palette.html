Usage Guidelines

  • Primary Red (#E63946): Use for CTAs, important buttons, active states, and brand emphasis
  • Primary Blue (#1D3557): Use for navigation, headers, and professional elements
  • Accent Blue (#457B9D): Use for links, interactive elements, and secondary buttons
  • Status Colors: Use consistently for success, warning, danger, and info messages
  • Gradients: Use for depth and modern appeal on buttons and headers
  • Neutral Colors: Use for text hierarchy and backgrounds
